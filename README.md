# 《阴间剧院》- 原创漫剧项目

## 📖 作品简介

**类型**：恐怖 / 奇幻 / 单元剧  
**格式**：动态漫剧（每集 3-5 分钟）  
**目标平台**：B 站、抖音、小红书  

---

## 🎭 核心设定

### 故事背景

主角**林夜**继承了一座位于城市老旧街区的废弃剧院——**「夜来香大剧院」**。

剧院看似破败，实则连接着阴阳两界。每晚午夜，剧院会自动「营业」，观众不是活人，而是滞留人间的亡魂。

### 核心机制

**「演出系统」**：
- 剧院有一台老式放映机，每晚会显示一个「演出任务」
- 完成任务（排演特定剧目）可以「超度」对应的亡魂
- 每超度一个亡魂，剧院会解锁新的区域/道具/能力
- 亡魂可能成为剧院的「常驻演员」

### 差异化亮点

| 对比项 | 恐怖屋类 | 阴间剧院 |
|--------|----------|----------|
| 载体 | 鬼屋参观 | 舞台表演 |
| 互动 | 游客被吓 | 亡魂被超度 |
| 节奏 | 探险式 | 排练 + 演出 |
| 视觉 | 场景展示 | 舞台灯光 + 表演 |
| 情感 | 刺激为主 | 每个故事都有泪点 |

## 👥 主要角色

### 林夜（男主）
- **年龄**：24 岁
- **身份**：戏剧学院肄业，夜来香大剧院继承人
- **性格**：表面毒舌，内心柔软，对表演有执念
- **特长**：编剧、导演、即兴表演
- **背景**：爷爷是剧院上一任主人，失踪前把剧院留给他

### 苏青（女主）
- **年龄**：22 岁
- **身份**：戏剧学院在校生，林夜的学妹
- **性格**：元气满满，胆子大，不信邪
- **特长**：表演、化妆、道具制作
- **背景**：偶然闯入剧院，成为第一个「活人员工」

### 白老板（常驻演员 1）
- **身份**：1940 年代当红戏子，红衣厉鬼
- **性格**：傲娇，戏痴，对表演要求极高
- **能力**：附身演员临时提升演技、操控舞台道具
- **执念**：想在真正的舞台上唱完最后一出戏

### 小周（常驻演员 2）
- **身份**：90 年代剧场电工，意外触电身亡
- **性格**：老实人，技术宅，胆小但可靠
- **能力**：操控剧院电路、修复设备
- **执念**：想修好剧院那台坏掉的放映机

## 🏛️ 剧院结构

```
夜来香大剧院（初始解锁区域）
├── 观众席（300 座，破旧但可用）
├── 主舞台（核心演出区域）
├── 后台化妆间（苏青的工作室）
├── 道具仓库（小周的地盘）
└── 【待解锁】
    ├── 地下室（存放「特殊剧目」）
    ├── 阁楼（白老板的「休息室」）
    └── 放映室（神秘的核心区域）
```

## 🎬 制作规格

### 视觉风格
- **色调**：暗红 + 金色（复古剧院感）
- **比例**：16:9（横屏，适配 B 站）
- **帧率**：12fps（动态漫标准）

### 单集结构
```
0:00-0:30  开场钩子（高能/悬念）
0:30-2:30  主线剧情
2:30-3:30  情感高潮/反转
3:30-4:00  结尾伏笔 + 下集预告
```

### 配音配置
- 林夜：青年男声（略带沙哑）
- 苏青：少女音（元气）
- 白老板：戏曲腔 + 现代音混合
- 小周：憨厚中年音
