# 《治愈系恐怖游戏》- 原创漫剧项目

## 📖 作品简介

**类型**：恐怖 / 治愈 / 无限流 / 单元剧  
**格式**：动态漫剧（每集 4-5 分钟）  
**目标平台**：B 站、抖音、小红书  
**核心概念**：用恐怖的方式，讲治愈的故事

## 🎮 核心设定（融合两本书精髓）

### 灵感来源

| 元素 | 《我有一座恐怖屋》 | 《治愈系游戏》 | 本项目融合 |
|------|-------------------|---------------|-----------|
| 载体 | 鬼屋 | 游戏副本 | **恐怖游戏 APP** |
| 机制 | 完成任务扩建鬼屋 | 通关副本获得奖励 | **通关副本解锁治愈值** |
| 鬼怪 | 收集员工 | 对抗/超度 | **理解鬼魂执念，帮他们放下** |
| 风格 | 恐怖刺激 | 温暖治愈 | **恐怖外壳 + 治愈内核** |
| 情感 | 以牙还牙的公平 | 救赎与和解 | **每个恐怖故事都有温暖结局** |

### 故事背景

主角**陆离**是一名普通大学生，某天手机自动安装了一个名为**「治愈系游戏」**的 APP。

APP 图标是一个微笑的小太阳，点开后发现是一个恐怖游戏——但通关方式不是逃跑或战斗，而是**理解鬼魂的执念，帮他们完成心愿**。

每通关一个副本，陆离会获得「治愈值」，可以用来：
- 升级 APP 功能
- 解锁新副本
- 兑换现实中的小幸运（比如考试蒙对、抽卡 SSR）

但随着副本深入，陆离发现这些鬼魂的执念...似乎都和自己有关。

### 核心机制

**游戏界面**：
```
【治愈系游戏 v1.0】
当前等级：Lv.1 见习治愈师
治愈值：120
已通关副本：3/10

【今日副本】（每日刷新）
- 午夜公交（难度：★）
-  missing 的娃娃（难度：★★）
- 最后一班电梯（难度：★★★）

【商城】
- 护身符（50 治愈值）
- 真相碎片（100 治愈值）
- 时光回溯（500 治愈值，限用 1 次/月）
```

**通关条件**：
- ❌ 传统恐怖游戏：逃跑/战斗/解谜
- ✅ 治愈系游戏：理解 + 共情 + 完成执念

**失败惩罚**：
- 不会死，但会失去部分「情感记忆」（比如忘记某个重要的人）

## 👥 主要角色

### 陆离（男主）
- **年龄**：21 岁
- **身份**：普通大学心理学系大三学生
- **性格**：表面冷淡，内心柔软，擅长共情
- **特长**：观察细节、心理分析、倾听
- **背景**：小时候经历过一场事故，失去了一段记忆
- **秘密**：APP 似乎认识他，副本里的鬼魂都叫他「阿离」

### 小太阳（APP 助手）
- **形象**：一个像素风的微笑小太阳
- **性格**：元气满满，说话带颜文字 (✿◡‿◡)
- **功能**：新手引导、任务发布、商城管理
- **秘密**：似乎有自我意识，知道陆离的过去

### 苏晓（女主）
- **年龄**：20 岁
- **身份**：陆离的同班同学，校心理社成员
- **性格**：阳光开朗，好奇心强，不信邪
- **特长**：调查、推理、社交
- **背景**：偶然发现陆离在玩「奇怪的游戏」，主动加入
- **作用**：现实世界的支援，帮陆离调查副本背后的真相

### 神秘人「K」
- **身份**：未知，似乎是游戏的前任玩家
- **出现方式**：在某些副本中留下线索
- **目的**：引导陆离发现真相
- **台词**：「别相信小太阳」「找回你的记忆」「逃离游戏」

## 🏛️ 世界观设定

### APP 的来源

世界上存在一种名为「执念能量」的非物质存在，是亡魂未了心愿的凝聚。

某个神秘组织（暂定名「心灵研究所」）开发了「治愈系游戏」APP，目的是：
- 收集执念能量
- 帮助亡魂超度
- 筛选有特殊能力的人（如陆离）

### 治愈值体系

治愈值本质是「正向情感能量」，可以：
- 升级 APP 功能
- 兑换现实幸运（小概率事件）
- 保护用户免受负面能量侵蚀
- 后期：可以复活/拯救某人（终极功能）

### 记忆系统

陆离失去的记忆是关键主线：
- 小时候的事故不是意外
- 他和 APP 的创造者有关系
- 某些副本里的鬼魂是他认识的人
- 小太阳可能是某个人格化存在

## 🎨 视觉风格

### APP 界面设计
- **主色调**：暖黄色（小太阳主题）
- **图标**：像素风 + 圆角设计
