## 配音要求

### 主要角色

| 角色 | 声线 | 参考 |
|------|------|------|
| 陆离 | 青年男声，冷淡但温暖 | 折木奉太郎（冰菓） |
| 苏晓 | 元气少女音 | 藤原千花（辉夜大小姐） |
| 小太阳 | 元气少女音 + 颜文字 | 可爱吉祥物 |
| 夏晴 | 温柔少女音，带悲伤 | 面码（未闻花名） |
| 爸爸/K | 中年男声，沧桑慈爱 | 成熟父亲形象 |
| 陈明 | 中年男声，冷静→疯狂 | 反派科学家 |

### 配音导演备注

1. **陆离**：前期冷淡，后期逐渐温暖，最终坚定
2. **苏晓**：始终保持元气，但有关键情感爆发
3. **小太阳**：前期元气，中期严肃，后期虚弱
4. **陈明**：从冷静到疯狂，层次分明
