## 分镜绘制规范

### 镜头语言

| 镜头类型 | 用途 | 示例 |
|---------|------|------|
| 远景 | 建立场景、孤独感 | 陆离独自站立 |
| 中景 | 对话、关系 | 陆离与苏晓对话 |
| 近景 | 情感表达 | 角色面部表情 |
| 特写 | 关键道具、冲击 | 手机屏幕、信 |
| 俯拍 | 压抑、渺小 | 陆离独处 |
| 仰拍 | 威胁、阴谋 | 研究所大楼 |

### 色调规范

| 场景类型 | 主色调 | 辅助色 |
|---------|--------|--------|
| 现实日常 | 冷蓝灰 | 白色 |
| 副本恐怖 | 冷绿/紫 | 黑色阴影 |
| 治愈时刻 | 暖黄/橙 | 柔光 |
| 研究所 | 冷白 | 蓝色光晕 |
| 记忆闪回 | 暖黄褐 | 柔焦 |

### 转场规范

| 转场类型 | 用途 |
|---------|------|
| 硬切 | 场景切换 |
| 淡入淡出 | 时间流逝 |
| 溶解 | 记忆闪回 |
| 光圈 | APP 界面 |
| 光点消散 | 超度场景 |
