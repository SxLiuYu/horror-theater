**登场角色**：陆离、苏晓、小太阳、神秘人 K（首次提及）  
**场景**：大学校园、游乐园、宿舍

**画面**：

**手机屏幕**：

## 制作备注

### 视觉风格
- **色调**：
  - 现实场景：冷色调（疏离感）
  - 游乐园：暖黄色（回忆感）
  - 记忆闪回：柔焦 + 金色光晕
- **镜头**：
  - 摩天轮：仰拍 + 旋转镜头（眩晕感）
  - 记忆闪回：快速剪辑 + 模糊边缘
  - 双人对话：中景（关系建立）

### 配音要求
- **陆离**：青年男声，冷淡但温暖，独白要自然
- **苏晓**：元气少女音，好奇、坚定、勇敢
- **小太阳**：元气少女音，带颜文字语气，偶尔严肃
- **爸爸/K**：中年男声，低沉、神秘、慈爱

### BGM 建议
- 开场：轻快日常（校园生活）
- 陆离苏晓对话：温暖钢琴（关系建立）
- 摩天轮记忆：悬疑 + 悲伤弦乐（记忆冲击）
- 苏晓手机震动：电子音（APP 激活）
- 彩蛋：低沉电子（阴谋揭露）

### 时长控制
| 段落 | 目标时长 |
|------|----------|
| 开场（APP 通知） | 30 秒 |
| 转折（苏晓登场） | 60 秒 |
| 高潮（摩天轮记忆） | 90 秒 |
| 结尾（苏晓加入） | 60 秒 |
| 彩蛋（K 现身） | 20 秒 |
| **总计** | **约 4 分钟** |
