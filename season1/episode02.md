# 第 2 集《午夜公交》- 完整剧本

**时长**：约 4 分钟  
**登场角色**：陆离、小太阳（APP 助手）、公交司机、女孩（回忆/鬼魂）  
**场景**：末班公交车、公交站台、宿舍

## 制作备注

### 视觉风格
- **色调**：
  - 现实场景：冷色调（疏离感）
  - 小区楼道：昏暗黄绿（不安感）
  - 娃娃发光：暖白色（治愈感）
- **镜头**：
  - 陆离独处：远景、俯拍（孤独感）
  - 女人读信：特写（情感爆发）
  - 培养皿：仰拍（阴谋感）

### 配音要求
- **陆离**：青年男声，冷淡但温暖，独白要自然
- **小太阳**：元气少女音，带颜文字语气，彩蛋中切换为冷静声线
- **女人**：中年女声，悲伤、颤抖、释然
- **陈明**：中年男声，冷静、有威严
- **小太阳本体**：少女音，但没有元气，平静中带悲伤

### BGM 建议
- 开场：轻快日常（宿舍生活）
- 楼道探索：悬疑钢琴（不安感）
- 发现娃娃：柔和弦乐（悲伤）
- 女人读信：情感高潮（钢琴 + 弦乐）
- 超度场景：温暖治愈（释然感）
- 彩蛋：低沉电子（阴谋揭露）

### 时长控制
| 段落 | 目标时长 |
|------|----------|
| 开场（宿舍 + 教室） | 45 秒 |
| 转折（放学后） | 45 秒 |
| 高潮（小区 + 女人） | 90 秒 |
| 结尾（回宿舍） | 60 秒 |
| 彩蛋（研究所） | 20 秒 |
| **总计** | **约 4 分钟** |
