# 第 8 集《记忆》- 完整剧本

**时长**：约 5 分钟  
**登场角色**：陆离、苏晓、小太阳、小陆离（10 岁）、爸爸、白大褂研究员  
**场景**：副本（小时候的家）、宿舍

---

**手机屏幕**：
```

【主线副本已解锁】
```

## 制作备注

### 视觉风格
- **色调**：
  - 记忆场景：暖黄色（怀旧）
  - 实验室：冷白色（冰冷）
  - 现实：恢复正常色调
- **镜头**：
  - 小陆离玩耍：温馨中景
  - 实验室闪回：快速切换 + 特写
  - 陆离头痛：旋转镜头（眩晕感）

### 配音要求
- **陆离**：青年男声，冷淡→震惊→痛苦→迷茫
- **苏晓**：元气少女音→担忧→哭泣
- **小太阳**：元气少女音→虚弱→焦急
- **爸爸**：中年男声，慈爱→挣扎→痛苦
- **白大褂**：冷静、专业、冷漠
- **陈明**：中年男声，冷静、有威严

### BGM 建议
- 开场：悬疑电子（副本警告）
- 记忆场景：温暖钢琴（怀旧）
- 实验室闪回：紧张弦乐 + 电子（危机）
- 手术场景：悲伤大提琴（牺牲）
- 醒来场景：沉默→悲伤钢琴
- 彩蛋：低沉电子（阴谋）

### 时长控制
| 段落 | 目标时长 |
|------|----------|
| 开场（副本通知） | 30 秒 |
| 转折（进入副本） | 60 秒 |
| 高潮（记忆 + 闪回） | 120 秒 |
| 结尾（醒来） | 60 秒 |
| 彩蛋（研究所） | 20 秒 |
| **总计** | **约 5 分钟** |
